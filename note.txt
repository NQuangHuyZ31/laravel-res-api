Resfull API
	2xx -> success
	200 -> Oke -> GET/PUT?PATCH success
	201 create - POST success
	204 no content - DELETE success


	4xx Client Errors
	400 bad request -> invalid input
	401 Unthorized -> no token or invalid token
	403 Forbidden -> valid token but no access
	404 Not Found -> resource not found
	422 Unprocessable Entity -> validation error

	5xx Server Errors
	500 Internal Server Error -> something went wrong on the server

Not return HTML and plain text

paginate:
	example: /api/v1/products?page=1&size=10
filter:
	example: /api/v1/products?category=electronics&price[gt]=100
sort:
	example: /api/v1/products?sort=price,asc
